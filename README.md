# Channel OS v0.1

MVP-бот для владельцев Telegram-каналов.

## Что есть

- Главное меню
- Регистрация пользователя при `/start`
- Подключение канала через пересланный пост
- Список подключенных каналов
- Создание текстового черновика
- Выбор канала для публикации
- Публикация поста в канал
- Сохранение истории постов в базу
- Базовая аналитика

## Установка

```bash
npm install
```

Создай `.env` на основе `.env.example`:

```env
BOT_TOKEN=токен_от_BotFather
DATABASE_URL="file:./dev.db"
```

Сгенерируй Prisma Client и создай базу:

```bash
npx prisma generate
npx prisma migrate dev --name init
```

Запуск:

```bash
npm start
```

## Как подключить канал

1. Добавь бота администратором в Telegram-канал.
2. Дай право публиковать сообщения.
3. Перешли в бота любой пост из этого канала.
4. Канал появится в разделе `Мои каналы`.

## Важно

Telegram Bot API не дает боту точный список подписчиков и отписок обычного канала. Поэтому в v0.1 нет функции "кто подписался/отписался".
