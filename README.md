# Channel OS v0.1.1 BotHost Fixed

Эта версия сделана специально для BotHost.

## Что исправлено

- Полностью убрана Prisma
- Больше не нужна команда `prisma generate`
- Больше не нужны миграции
- Бот запускается обычной командой:

```bash
npm start
```

## Установка на BotHost

1. Загрузи проект на GitHub.
2. На BotHost подключи репозиторий.
3. В переменных окружения добавь:

```env
BOT_TOKEN=твой_токен_бота
```

4. Команда запуска:

```bash
npm start
```

## Функции v0.1

- Главное меню
- Подключение канала через пересланный пост
- Список подключенных каналов
- Создание текстового черновика
- Выбор канала для публикации
- Публикация поста в канал
- Базовая аналитика
